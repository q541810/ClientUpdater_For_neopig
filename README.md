## 客户端自动更新器_for猪起

额如你所见，这是一个用于自动更新客户端的工具，他的作用是更好的为猪起新世纪这个服务器服务 

因为原版停更太久而且用起来有那么一捏捏的难受，所以说我就干脆自己开一个分支单独维护了 

啊就是这样，多说无益 

反正原的仓库链接是[ClientUpdater](https://github.com/XiaoJiang0208/ClientUpdater)  
仓库的readme文件是[原仓库的中文版本readme](https://github.com/XiaoJiang0208/ClientUpdater?tab=readme-ov-file#chinese)

使用教程可以参考原仓库的

并且因为我把config文件的名字设置的和原仓库的模组是一样的，所以使用原仓库版本的可以直接无缝迁移上来！
